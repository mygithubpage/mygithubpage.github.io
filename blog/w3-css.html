W3 Tooltip

Block Tooltip use block elements like div to enlarge the hoverable area. Include w3-text in w3-tooltip and use w3-panel to increase readability. Inline Tooltip used html global attribute title.

ToolTip
Text 

            <div class="w3-tooltip">ToolTip 
              <div class="w3-text w3-light-gray w3-panel">
                Text
              </div>
            </div>
